# 🌩️ AWS Cloud Foundations

This repository documents my learning journey through AWS Cloud Foundations. It includes core cloud concepts, AWS services, and hands-on labs based on real-world scenarios and AWS training modules.

---

## 🎯 Objectives

- Understand essential cloud computing principles  
- Gain hands-on experience with core AWS services  
- Design basic cloud architectures following best practices  
- Build a strong foundation for cloud security and DevSecOps  
- Prepare for AWS certifications (Cloud Practitioner & beyond)

---

## 🧠 Topics Covered

### ☁️ Cloud Computing Fundamentals
- What is Cloud Computing (IaaS, PaaS, SaaS)
- Cloud deployment models (Public, Private, Hybrid)
- Benefits of cloud (scalability, elasticity, high availability)

### 💰 Billing & Pricing
- AWS Pricing Models (On-Demand, Reserved, Spot Instances)
- Cost optimization strategies
- AWS Free Tier usage
- Billing Dashboard & Cost Explorer

### 🌍 AWS Global Infrastructure
- Regions & Availability Zones
- Edge Locations & CDN (CloudFront)
- High availability design principles

### 🔐 Identity & Security (IAM)
- IAM Users, Groups, Roles
- Policies & Permissions
- Shared Responsibility Model
- MFA & Best Practices

### 🌐 Networking
- VPC (Virtual Private Cloud)
- Subnets (Public & Private)
- Internet Gateway & NAT Gateway
- Route Tables & Security Groups
- Network ACLs

### 🖥️ Compute Services
- EC2 (Elastic Compute Cloud)
- EC2 Instance Types & AMIs
- Auto Scaling Groups
- AWS Lambda (Serverless)
- Elastic Beanstalk (PaaS)

### 📦 Storage Services
- S3 (Object Storage)
- S3 Storage Classes
- EBS (Block Storage)
- EFS (File Storage)
- Glacier (Archival Storage)

### 🗄️ Databases
- RDS (Relational Databases)
- DynamoDB (NoSQL)
- Database scaling & backups
- Multi-AZ & Read Replicas

### 📊 Monitoring & Management
- CloudWatch (Metrics, Logs, Alarms)
- CloudTrail (Auditing)
- AWS Config
- Trusted Advisor

### ⚖️ Scaling & High Availability
- Load Balancers (ALB, NLB)
- Auto Scaling strategies
- Fault tolerance design

### 🏗️ Cloud Architecture
- Well-Architected Framework
- Best practices (Security, Reliability, Performance, Cost Optimization)
- Basic 3-tier architecture
